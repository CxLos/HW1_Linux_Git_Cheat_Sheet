`git init` - initialize existing directory as git repository
`git clone [url]` - retrieve a remote repository from a hosted location to your local pc
`git status` - show modified files in working directory
`git add` - add a file as it is now to next commit
`git branch` - list all your branches
`git branch [branch_name]` - create new branch
`git checkout [branch_name]` - switch to branch
