`git init` - initialize existing directory as git repository
`git clone [url]` - retrieve a remote repository from a hosted location to your local pc
`git status` - show modified files in working directory
`git add` - add a file as it is now to next commit
`git branch` - list all your branches
`git branch [branch_name]` - create new branch
`git checkout [branch_name]` - switch to branch
`git checkout -b [branch_name]` - create new branch and switch to it
`git merge [branch]` - merge the specified branch history in the current one
`git log` - show the commit history for current active branch
`git show [SHA]` - show any object in git in human-readable format
`git rm [file]` - delete the file from project and stage the removal for commit
`git mv [existing_path] [new_path]` - change an existing file path and stage the move
`git stash` - save modified and staged changes
